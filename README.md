# Luxury Hotel Website For Kuriftu Hotel

A modern and responsive hotel website designed and developed to showcase hotel services, rooms, dining experiences, reservations, and company information. This project was built as a front-end web development project using HTML, CSS, and JavaScript.

The website provides visitors with an attractive and user-friendly interface where they can explore hotel features, browse rooms, make reservations, and learn more about the hotel.



## Features

* Responsive and modern website design
* Home page with hotel introduction
* Room showcase section
* Dining and restaurant information page
* Reservation form page
* About page
* News and updates section
* Interactive user interface using JavaScript
* Organized file and folder structure



## Technologies Used

* HTML5
* CSS3
* JavaScript

## Installation and Usage

1. Clone the repository:

```bash
git clone https://github.com/DuDu21cs/web_hotel.git
```

2. Open the project folder:

```bash
cd web_hotel
```

3. Run the project:

* Open `main.html` in your browser.
* Or use the VS Code Live Server extension for a better development experience.
